                            // Get the string the user entered
                            function getString(){
                                
                                // Make the alert box invisible if it is not already
                                document.getElementById("alert").classList.add("invisible");
                                
                                // Get the string the user entered
                                let userString = document.getElementById("userString").value;

                                // Check if string is valid
                                if (validString(userString)) {

                                    // Get the reversed string
                                    let revString = reverseString(userString);
                                    
                                    // Display the reversed string
                                    displaySuccess(revString);
                                }
                                else {

                                    // If the string has less than 2 characters then display an error
                                    displayError();
                                }
                            }

                            // Check if the given string is valid (not just white space)
                            function validString(userString){
                                
                                if (userString.trim().length > 1)
                                {
                                    return true;
                                }
                                else {
                                    return false;
                                }
                            }

                            // Reverse the given string
                            function reverseString(userString){

                                let revString = []
                                for (let index = userString.length - 1; index >= 0; index--) {

                                    revString += userString[index];
                                }

                                return revString;
                            }

                            // Change color of the alert
                            function setAlertColor(alertClass){

                                switch (alertClass){
                                    case "alert-success":
                                        document.getElementById("alert").classList.remove("alert-danger");
                                        document.getElementById("alert").classList.add("alert-success");
                                        break;
                                    case "alert-danger":
                                        document.getElementById("alert").classList.remove("alert-success");
                                        document.getElementById("alert").classList.add("alert-danger");
                                }
                            }

                            // Display success message and reversed string
                            function displaySuccess(revString){

                                setAlertColor("alert-success");
                                document.getElementById("msgHead").innerHTML = `Success!`;
                                document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`
                                document.getElementById("alert").classList.remove("invisible");
                            }

                            // Display error message 
                            function displayError(){
                                
                                setAlertColor("alert-danger");
                                document.getElementById("msg").innerHTML = "";
                                document.getElementById("msgHead").innerHTML = `Error : You must enter a valid string of at least 2 characters.`;
                                document.getElementById("alert").classList.remove("invisible");
                            }

The Code is structed into multiple functions, the main focus in this project was the use of the arrays. But a switch statement was used as well.

getString()

This is the main function, it starts off by setting the alert box to be hidden. After that it gets the input from the user and checks if it is valid with the validString() function. If the string is valid, it is then reversed with the reverseString() function. Lastly either a success or error alert box is displayed with the reversed string.

validString()

This function takes a string as an argument, the string is then trimmed. Trimming the string removes empty whitespace in front and behind the string. After the string is trimmed the length is checked ensure it contains more than 1 character. The trim is nessary here to avoid reversing a string full of spaces and no characters. The length is checked to make sure the string contains more than 1 character, anything 1 or less cannot be reversed.

reverseString()

This function takes a string as an argument, then reverses it. This is done by first creating a new array to hold the newly reversed string revString. Then a for loop is ran starting from the from the ending index and working back to 0 each letter is added to revString. The fully reversed string is then returned.

setAlertColor()

This function take a string as an argument, the string it takes it the class name of "alert-success" or "alert-danger". Those are both class names that control the color of the alert box. The function uses a switch statement, it will remove the opposing class and add the requested class. In this case changinng the alert box from red to green and vise versa.

displaySuccess()

This function has the reversed string passed to it. The alert color is set to success via the setAlertColor() function. Then the alert header message to success and the alert message content is set to the reversed string. Lastly the alert box is unhidden to be seen by the end user.

displayError()

This function is called in the event the string is not valid. It sets the alert color via the setAlertColor() function. Then the alert header is set to an error message and the message content is cleared. Lastly the alert box is unhidden to be seen by the end user.
